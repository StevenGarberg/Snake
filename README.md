# Snake
A remake of the classic game Snake with Blazor WASM.

https://blazorsnake.z13.web.core.windows.net/

## Controls
| Keys | Description |
| ------------- | ------------- |
| `a` / `⬅` | Change direction to left |
| `d` / `➡` | Change direction to right |
| `w` / `⬆` | Change direction to up |
| `s` / `⬇` | Change direction to down |
